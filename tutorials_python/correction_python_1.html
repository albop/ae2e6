Manipulating AR1 Models

We consider here a simple autoregressive model:

\[y_t = A y_{t-1} + B e_t\]

where \(y_t=(y^1_t, y^2_t)\) a vector of variables and \(e_t=(e^1_t, e^2_t)\) a normal i.i.d. multivariate process defined by covariance matrix \(\Sigma \in R^p \times R^p\).

We start by choosing:

\[A = \begin{bmatrix}\rho & 2 \\ 0 & \lambda\end{bmatrix}\]

\[B = \begin{bmatrix}1 & 0 \\ 0 & 1\end{bmatrix}\]

\[\Sigma = \begin{bmatrix}0.05 & 0.005 \\ 0.005 & 0.01\end{bmatrix}\]

Define python variables for matrices \(A, B, \Sigma\).

import numpy as np

# Parameters

λ = 0.5

# Matrices

              [0.0, λ]])
# B is the identity matrix
np.eye(2, dtype=float) # pay attention to use floats rather than integers
Σ= np.array([[0.05,  0.005],
                  [0.005, 0.01]], dtype=float)

n_std = 2
N = 1000

# --- random draws: equivalent to random_points = [rand(dis) for i=1:1000] ---

# --- helper: covariance ellipse like covellipse! ---
def add_cov_ellipse(ax, mean, cov, n_std=2, **kwargs):
    # eigen-decomposition
    vals, vecs = np.linalg.eigh(cov)
    order = vals.argsort()[::-1]
    vals, vecs = vals[order], vecs[:, order]

    # angle of the largest eigenvector (degrees)
    angle = np.degrees(np.arctan2(vecs[1, 0], vecs[0, 0]))

    # ellipse radii = n_std * sqrt(eigenvalues)
    width, height = 2 * n_std * np.sqrt(vals)

    ell = Ellipse(xy=mean, width=width, height=height, angle=angle, fill=False, **kwargs)
    ax.add_patch(ell)
    return ell


fig, ax = plt.subplots(figsize=(5, 5))

def simulate(A, B, Σ, e0=None, T=20, rng=None):
   
  #  A = np.asarray(A, dtype=float)
   # B = np.asarray(B, dtype=float)
    

    n = A.shape[0]   #dimension de y_t
    m = Σ.shape[0]   #dimension des chocs

    if e0 is None:
        e0 = np.zeros(m, dtype=float)
    else:
        e0 = np.asarray(e0, dtype=float)

    if rng is None:
        rng = np.random.default_rng()

    # sim = [B*e0]
    y = B @ e0
    sim = [y]

    for _ in range(T):
        # e1 = rand(MvNormal(Σ))
        e1 = rng.multivariate_normal(mean=np.zeros(m), cov=Σ)

        # y1 = A*y0 + B*e1
        y = A @ sim[-1] + B @ e1

        sim.append(y)

    # return hcat(sim...)
    return np.column_stack(sim)

Ergodic Distribution.

It can be shown that the ergodic distribution of a VAR1 is a multivariate normal law, with covariance matrix \(\Omega\).

This matrix is a solution to the equation \(\Omega = A \Omega A' + B \Sigma B'\).

A simple algorithm to find it consist in applying the recurrence \(\Omega_n = A \Omega_{n-1} A' + B \Sigma B'\) until convergence, starting with \(\Omega_0 =0\).

Implement this algorithm.

def ergodic_steady_state(A, B, Σ, N=1000, tol_eta=1e-8):
    
  #  A = np.asarray(A, dtype=float)
  #  B = np.asarray(B, dtype=float)
  #  Σ = np.asarray(Σ, dtype=float)

    #initilisation Omega0
    n = A.shape[0]
    Omega0 = np.zeros((n, n), dtype=float)

    for _ in range(N):
        Omega = A @ Omega0 @ A.T + B @ Σ @ B.T
        eta = np.max(np.abs(Omega - Omega0)) #critère d'arret
        if eta < tol_eta:
            return Omega
        Omega0 = Omega

    raise RuntimeError("No convergence")
